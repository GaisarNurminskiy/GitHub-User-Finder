# GitHub User Finder

## Автор
Гайсар Исмагилов

## Описание
Простое десктопное приложение с графическим интерфейсом (GUI) для поиска пользователей на платформе GitHub. Позволяет находить профили по логину, просматривать основную информацию (имя, аватар) и добавлять пользователей в локальное избранное.

## Как использовать API
Приложение использует публичный REST API GitHub. Для получения данных отправляется GET-запрос на `https://api.github.com/users/{username}`. Ответ обрабатывается в формате JSON.

## Примеры использования
1. Запустите приложение.
2. Введите логин (например, `torvalds`) и нажмите **«Поиск»**.
3. Просмотрите информацию о пользователе.
4. Нажмите **«Добавить в избранное»**.
5. Нажмите **«Показать избранное»** для просмотра сохраненных профилей.
